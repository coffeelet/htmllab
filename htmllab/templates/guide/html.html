HTML 快速指南

超文本标记语言 (HyperText Markup Language) 基础

基础骨架

每个 HTML 页面都应该包含以下基础结构：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <h1>你好，世界！</h1>
    <p>这是一个段落。</p>
</body>
</html>

常用标签

标题与文本

  • <h1> 到 <h6>: 标题（1级最大，6级最小）
  • <p>: 段落
  • <strong>: 加粗文本
  • <em>: 斜体文本
  • <br>: 换行

链接与图片

  • <a href="url">链接文本</a>: 超链接
  • <img src="image.jpg" alt="描述文字">: 图片

列表

<!-- 无序列表 -->
<ul>
    <li>列表项 1</li>
    <li>列表项 2</li>
</ul>

<!-- 有序列表 -->
<ol>
    <li>第一步</li>
    <li>第二步</li>
</ol>

语义化标签

使用具有明确意义的标签可以让搜索引擎和辅助设备更好地理解你的页面：

  • <header>: 头部区域
  • <nav>: 导航链接
  • <main>: 主要内容区域
  • <section>: 章节或节
  • <article>: 独立的文章内容
  • <aside>: 侧边栏（次要内容）
  • <footer>: 页脚区域
