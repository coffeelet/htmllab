颜色表示法

在 CSS 中，有四种主要方式表示颜色：

1. 颜色名称 (Named Colors)

color: red;
background-color: blue;
border-color: green;

CSS 定义了 140+ 个标准颜色名称

2. 十六进制 (HEX)

color: #ff0000;      /* 红色 */
color: #f00;         /* 简写 */
color: #ff000080;    /* 带透明度 */

#RRGGBB 格式，两两一对表示红绿蓝

3. RGB / RGBA

color: rgb(255, 0, 0);         /* 红色 */
color: rgba(255, 0, 0, 0.5); /* 半透明红 */
color: rgb(255 0 0 / 50%);   /* 新语法 */

0-255 表示红绿蓝，A 表示透明度 (0-1)

4. HSL / HSLA

color: hsl(0, 100%, 50%);       /* 红色 */
color: hsla(0, 100%, 50%, 0.5); /* 半透明 */

H=色相(0-360), S=饱和度(%), L=亮度(%)

💡 小贴士：HEX 最常用，HSL 最适合调整颜色（容易理解亮度/饱和度）

配色使用技巧

60-30-10 法则

  • 60% - 主色（背景、大面积）
  • 30% - 次要色（区块、卡片）
  • 10% - 强调色（按钮、链接）

对比度原则

  • 深色背景用浅色文字
  • 浅色背景用深色文字
  • 保证足够的可读性对比
  • 可用在线工具检查对比度
💡 颜色心理学：
红色 = 激情、紧急 | 蓝色 = 专业、信任 | 绿色 = 自然、安全
黄色 = 活力、注意 | 紫色 = 奢华、创意 | 黑色 = 高端、神秘
