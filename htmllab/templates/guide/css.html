CSS 核心属性指南

Cascading Style Sheets - 掌握网页样式与布局的艺术

1. 基础语法与引入

CSS 由选择器和声明块组成：

/* 选择器 { 属性: 值; } */
.my-class {
    color: #333;
    font-size: 16px;
}
💡 优先级： 内联样式 (style="") > ID 选择器 > 类选择器 > 标签选择器。

2. 文本与字体属性 (Text & Fonts)

color 文字颜色 (如: red, #fff, rgb(0,0,0))
font-size 字体大小 (如: 16px, 1.2rem, 150%)
font-family 字体族 (如: "Arial", sans-serif)
font-weight 字体粗细 (如: bold, 400, 700)
text-align 水平对齐 (left, center, right, justify)
line-height 行高，用于控制文字行间距
text-decoration 文本装饰 (none, underline, line-through)
letter-spacing 字符间距 (如: 2px)

3. 盒子模型 (Box Model)

所有 HTML 元素都可以看作盒子。它是布局的核心：

Margin (外边距)
Border (边框)
Padding (内边距)
Content (内容)
width / height 设置内容的宽度和高度
padding 内容与边框之间的透明区域
border 围绕在内边距和内容外的边框
margin 清除边框外的区域，它是透明的
💡 建议设置： box-sizing: border-box; 可以让 padding 和 border 不会增加盒子的总宽度。

4. 背景与边框效果

background-color 设置背景颜色
background-image 设置背景图片 (url('...'))
background-size 图片大小 (cover, contain, 100%)
border-radius 设置圆角 (如: 8px, 50%)
box-shadow 添加阴影 (如: 0 4px 8px rgba(0,0,0,0.1))
opacity 透明度 (0 到 1 之间)

5. 布局与定位 (Layout & Position)

Display 属性

  • block: 独占一行（如 <div>, <p>）
  • inline: 随文字排列，不接受宽高（如 <span>, <a>）
  • flex: 弹性布局，最常用的布局方式。
  • grid: 网格布局，用于复杂的二维布局。
  • none: 隐藏元素且不占用空间。

Position 属性

  • static: 默认值，按照正常文档流排列。
  • relative: 相对定位，相对于原本位置移动。
  • absolute: 绝对定位，相对于最近的非 static 父元素。
  • fixed: 固定定位，相对于浏览器窗口。
  • sticky: 粘性定位，滚动到特定位置时固定。

6. Flexbox 弹性布局 (必学)

应用于父元素（容器）的常用属性：

.container {
    display: flex;
    flex-direction: row;      /* 主轴方向：水平 */
    justify-content: center;  /* 主轴对齐：居中 */
    align-items: center;      /* 交叉轴对齐：居中 */
    flex-wrap: wrap;          /* 换行控制 */
    gap: 20px;                /* 子元素间距 */
}
