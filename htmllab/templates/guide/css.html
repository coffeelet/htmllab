CSS 快速指南

层叠样式表 (Cascading Style Sheets) 样式与布局

基础语法

/* 选择器 { 属性: 值; } */
p {
    color: blue;
    font-size: 16px;
    text-align: center;
}

选择器

  • h1: 标签选择器（匹配所有 <h1>）
  • .classname: 类选择器（匹配 class="classname"）
  • #idname: ID 选择器（匹配 id="idname"）
  • a:hover: 伪类（鼠标悬停在链接上时）

盒子模型 (Box Model)

每个 HTML 元素都可以看作一个盒子：

Margin (外边距)
Border (边框)
Padding (内边距)
Content (内容)
  • width / height: 内容宽高
  • padding: 内边距（内容与边框之间）
  • border: 边框
  • margin: 外边距（元素与其他元素之间）

布局 (Flexbox)

最常用的现代布局方式：

.container {
    display: flex;
    justify-content: center; /* 水平居中 */
    align-items: center;     /* 垂直居中 */
    flex-direction: row;      /* 排列方向 */
}
