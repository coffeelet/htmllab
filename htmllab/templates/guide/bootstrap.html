Bootstrap 5 快速指南

最流行的前端 CSS 框架入门指南

什么是 Bootstrap？

Bootstrap 是一个免费开源的前端框架，提供了丰富的 CSS 和 JavaScript 组件，帮助你快速构建响应式网页。

💡 核心优势：无需从零编写 CSS，使用预定义类即可创建美观界面

1. 栅格系统（布局核心）

Bootstrap 使用 12 列栅格系统，通过 row 和 col 类创建响应式布局。

基础用法

<div class="container">
    <div class="row">
        <div class="col-md-6">占一半宽度</div>
        <div class="col-md-6">占一半宽度</div>
    </div>
</div>

断点说明

断点 类前缀 尺寸
手机col-< 576px
平板col-sm-≥ 576px
桌面col-md-≥ 768px
大屏col-lg-≥ 992px

3. 实用工具类

Bootstrap 提供了大量工具类用于快速调整样式：

间距 (Spacing)

<div class="m-3">四周外边距 1rem</div>
<div class="mt-5">上方外边距 3rem</div>
<div class="p-2">四周内边距 0.5rem</div>
<div class="px-4">左右内边距 1.5rem</div>

格式: {property}{sides}-{size}
m=margin, p=padding | t=上, b=下, x=左右, y=上下 | 0-5 数字越大间距越大

颜色与背景

主色 成功 危险 警告 信息 
<span class="text-primary">蓝色文字</span>
<div class="bg-success text-white">绿色背景白字</div>

文字对齐与显示

<p class="text-center">居中文字</p>
<p class="text-end">右对齐</p>
<p class="fw-bold">粗体</p>
<p class="text-muted">灰色次要文字</p>
<div class="d-flex justify-content-between">Flex 布局</div>

4. 响应式设计

<!-- 只在移动端显示 -->
<div class="d-block d-md-none">手机可见</div>

<!-- 只在桌面显示 -->
<div class="d-none d-md-block">桌面可见</div>

<!-- 响应式文字大小 -->
<h1 class="display-4 display-md-3">标题</h1>

5. 常用类名速查

布局

  • .container - 固定宽度容器
  • .container-fluid - 全宽容器
  • .row - 行
  • .col-* - 列
  • .d-flex - Flex 布局

组件

  • .btn - 按钮基础
  • .card - 卡片
  • .alert - 警告框
  • .badge - 徽章
  • .navbar - 导航栏
